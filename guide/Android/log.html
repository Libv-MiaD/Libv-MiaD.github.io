# 2020.5.0lts

由于安卓更换了云备份的特殊性，使用喵滴自带云备份的小伙伴（webdav不受本次更新影响） 在更新2020.5.50beta版本时需要做以下注意：

  1. 用户信息会清除，需要重新登录
  2. 需要设置重新备份数据到新数据库中，若有部分数据在原来的云数据库中，小伙伴可以到官网先下载2020.5以下的旧版本，下载=>入口，把所有数据都下载回来，步骤:个人信息->高级会员->拉取，而后再更新2020.5以上版本，而后把数据重新备份一次，步骤:设置->实验室->重新进行备份，退出App，再进入App，数据就可以备份完成了。
  3. 同一书本和章节下，不可以出现相同的文章名称，否则云备份时会进行覆盖。

# 2020.4.1

下载更新以后可能文章找不到了，不要急，先选择默认书本的默认目录看看能不能找到？（删了的话，就重新建，缺哪个建哪个）。还找不到？不要急，创建一个书本叫Temp，在书本下创建一个目录null，这样所有数据都能找回了。
