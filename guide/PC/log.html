# 喵滴 Win/Mac/Linux 更新记录

# 2021.1.1更新

本次更新支持Windows/Mac/Linux，Windows需要卸载原版本再安装，否则会出现两个喵滴App

  • 修复:无法保存文件
  • 修复:字数统计卡片与预览按钮重叠
  • 添加:自定义级文件夹存储
  • 添加:文件夹读取与操作【重点】
  • 优化:文章上传与更新
  • 优化若干细节

# 2021.1.0更新

本次更新支持Windows/Mac/Linux

  • 修复:若干小细节
  • 修复:无法选择背景图片
  • 优化:主题选择界面
  • 添加:若干bug

# 2020.1.19更新注意事项

下载更新以后可能文章找不到了，不要急，先选择默认书本的默认目录看看能不能找到？（删了的话，就重新建，缺哪个建哪个）。

  • 全局字体

  • 支持markdown文件快速导入、导出
  • 优化保存机制
  • 支持书架模式
  • 全新云备份模式
