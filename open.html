# 开放平台

实验功能，请勿过度依赖

喵滴的高级用户可以自己动手搭建功能，自行调用喵滴的开放接口 目前已经实现的开放能力有：添加文章

# 文档

# 添加文章 接入文档

  1. 到https://api.miaodiapp.com/api/index.html申请喵滴key，并且妥善管理好。长度约，如85H7f36H2Rv2u3fYk9vLiQz

  2. 接入添加文章接口

curl --location --request POST 'https://sunshinesudio.club:8012/page/open/add' \
--form 'key="开放码"' \
--form 'Title="标题"' \
--form 'Content="内容"' \
--form 'Book="书本名"' \
--form 'Chara="目录名"'

其中所有字段都必须填写,同一个书本以及目录下面，不能出现相同的文章标题。

返回添加成功结果

{
    "info": "xxxx",//文章id
    "code": 200
}
  1. 官方实例

该功能仅提供高级会员使用

  • 到https://api.miaodiapp.com/api/index.html申请喵滴key，并且妥善管理好。长度约，如85H7f36H2Rv2u3fYk9vLiQz
  • 关注喵滴公众号Libv, 直接输入短语:key:你的key 例如:key:85H7f36H2Rv2u3fYk9vLiQz
  • 绑定成功后，每次发送文字，会自动保存到喵滴的备份库中。可实现不打开喵滴App存储文字。
  • 文字格式1: 纯内容,自动保存到wechat/默认文件夹下。
  • 文字格式2: 标题||内容
  • 文字格式3: 标题||内容||书本
  • 文字格式4: 标题||内容||书本||目录
