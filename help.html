# 喵滴使用帮助

# 1.文章保存

Android/iOS

  1. 喵滴App无需手动按保存，文本一旦有所变化，所有数据均会保存下来。几乎不存在文章丢失的情况。
  2. 可以根据文章列表页每个文章右下角的云朵☁️是否完整来判断是否已经备份到云端。不完整的小云则意味着没有进行同步，此时若出现App无法打开，则数据是无法进行读取的。因此强烈建议大家配置好云端备份或者激活喵滴会员。

PC
文章需要手动按ctrl+s进行保存，PC版暂时没有及时保存的功能。

# 2.Webdav使用

支持Android/iOS

  1. 使用前，请在Webdav服务器中创建一个名为Markdown的根目录。
  2. 任何的喵滴文章，都需要存在Markdown文件夹中的两层文件夹下，比如Markdown>2021年>2月>具体的md文件。只有按这种格式操作，喵滴App才可读取到文章哦！
  3. 若Webdav是使用坚果云，请注意使用应用密码，而不是账号密码进行Webdav登录。
  4. 自动同步功能需要开启会员后才能使用，普通用户需手动拉取【步骤：用户信息->高级会员服务->拉取文章备份】

# 3.关于会员激活

支持Android/iOS

  1. 下载喵滴客户端(Android：2020.5版本以上)
  2. 登录或注册
  3. 用户信息->激活码激活->输入激活码
  4. 显示activated success即为激活成功
  5. 重启客户端刷新会员状态

激活注意⚠️事项

  1. 激活码只能激活一次
  2. 体验邀请码可激活多次

# 4.使用模式

支持Android

喵滴App默认开启Markdown所见即所得预览。若不使用Markdown，可使用写作模式。在该模式下，段落与段落之间有清晰的分割，使小伙伴们更容易分辨。【开启方式:编辑页->弹出键盘->快捷栏左边小猫->编辑器设置->☑️写作模式】
